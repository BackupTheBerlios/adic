ADIC

Any door is closed.

Description

ADIC (Any door is closed) is a unique cooperative multi-player game. The goal is to lock all the players of the other teams. Main development platform is GNU/Linux - but now there are cross-compiled binaries for win32. Beside of playing the game you can write a bot and use the game as test bed for AI/robotics algorithms.

Running

  • Take a look at the meta-server to find a running server
  • if the list is empty start your own server. Run adicserver (perhaps you want to enable the metaserver and choose another map. f.e: adicserver --useMetaServer=1 --meshURI=data:mesh.xml, see adicserver --help for more info)
  • run ./adicclient (perhaps you need to set --libGL and perhaps you want to set --quality=3 if you have hardware accelerated OpenGL, run adicclient --help for more info)
  • to start a bot team run adicbot --implementation=i (Where i is currently "KarmeBot" or "BlackBot")

Gameplay

Goal

ADIC is a cooperative multiplayer game. The goal is to lock all players of the other teams. A player is locked if all doors of the room he is in are closed. A locked player can't move.

The characters

The characters differ in speed, mass, endurance, ...

Fitness

Either embrace your team mates, drink some water from the springs or just take a rest to recover.
