ADIC

Any door is closed.

Description

ADIC (Any door is closed) is a cooperative multi-player game. The goal is to lock all the players of the other teams. Main development platform is GNU/Linux - but now there are cross-compiled binaries for win32.

  • DOPE++ (and of course its requirements)
  • SDL >=1.2
  • SDL_mixer >=1.2
  • SDL_image >=1.2
  • OpenGL or Mesa

Running

  • Take a look at the meta-server to find a running server
  • if the list is empty start your own server. Run ./adicserver (perhaps you want to disable the metaserver and choose another map. f.e: ./adicserver --metaServer="" --meshURI=data:mesh.xml, see adicserver --help for more info)
  • run ./adicclient (perhaps you need to set --libGL and perhaps you want to set --quality=1 if you have hardware accelerated OpenGL, see adicclient --help for more info)

Gameplay

Goal

ADIC is a cooperative multiplayer game. The goal is to lock all players of the other teams. A player is locked if all doors of the room he is in are closed. A locked player can't move.

The characters

The characters differ in speed, mass, endurance, ...

Fitness

Either embrace your team mates, drink some water from the springs or just take a rest to recover.
